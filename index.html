🚀 快速上手

from gaskill import Complex, Frac, vec2

# 复数 — 支持字符串解析
c = Complex("3+4j")
print(c * c) # (-7+24i)

# 分数 — 精确有理数
f = Frac(1, 3) + Frac(1, 6)
print(f) # 1/2

# 2D 向量 — 几何运算
v = vec2(3, 4)
print(v.length()) # 5.0

📦 安装

pip install mlib 或直接复制源码到项目中

📚 模块总览

smath
数学核心：复数、素数、伽马函数、三角函数
sfrac
精确分数运算，支持百分数、LaTeX 输出
svector / smatrix / squaternion
向量、矩阵、四元数 — 全套线性代数
sgeometry / sgeometry3d
2D / 3D 碰撞检测、射线、AABB、多边形
spathfinder
A* · Dijkstra · BFS · 双向搜索 · 迷宫生成
scrypt
RSA 加解密 · Shamir 秘密共享 · 自定义加密
hashlib / ecc
SHA-256 · DJB2 · ECDSA 签名 (secp256k1)
sbase64
Base16/32/64/58/256（中文汉字版）
scache / spool
LRU/LFU/TTL 缓存 · 对象池 · 环形缓冲区
sui
轻量 UI 框架：标签、按钮、输入框、进度条
ssaver / database
SGT 存档器 · 磁盘 KV 数据库（纯 Python）
snoise / scurve / seasing
Perlin/Simplex 噪声 · 贝塞尔/B样条 · 缓动函数
sformat / re
表格生成 · 时间/文件大小格式化 · 正则引擎
poly / radical
多项式因式分解 · 根式化简与四则运算
bmp / pickle / zip
BMP 读写 · 二进制序列化 · LZ77+哈夫曼压缩
